    OMETIFF.next — Method.
    next(file::TiffFile)

    Loads the next IFD in file as determined by file.loc and returns a list of the strip offsets to load the data stored in this IFD. If the IFD has not been loaded yet then this function loads it from disk, otherwise it returns the IFD location from memory.
