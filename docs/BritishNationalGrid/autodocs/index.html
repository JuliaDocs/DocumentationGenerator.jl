    BritishNationalGrid.gridref — Function.
    gridref(p::BNGPoint, n, square:false, separator=" ")

    Return a string giving an n-figure grid reference. By default, a full reference is given. If square is true, then supply the 100 km square name first, then the reference within that square. The square, eastings and northings are separated by separator.

    julia> gridref(BNGPoint(429157, 623009), 8, true, separator="_")
"NU_2915_2300"

    BritishNationalGrid.lonlat2bng — Method.
    lonlat2bng(lon, lat) -> easting, northing
lonlat2bng(lon::AbstractArray, lat::AbstractArray) -> A::Array

    Transform from longitude and latitude in WGS84 into BNG easting and northing (m). The first form does so for scalars and returns a tuple; the second form does so for length-n arrays and returns a n-by-2 array where the first column is the easting, and the second is the northing.
